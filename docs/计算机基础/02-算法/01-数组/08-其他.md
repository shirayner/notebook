# 其他

#### [【238】 除自身以外数组的乘积](https://leetcode-cn.com/problems/product-of-array-except-self/)

方法：左右乘积列表

思路：算好左右乘积列表，然后根据列表来计算最终结果，达到O(n)的目的

题解：

```java
class Solution {

    /**
     * 方法一：左右乘积列表
     * <p>
     * 时间 O(n)，空间 O(n)
     *
     * @param nums nums
     * @return 除 nums[i] 之外其余各元素的乘积。
     */
    public int[] productExceptSelf(int[] nums) {
        int size = nums.length;

        // 提前算好左右乘积
        int[] L = new int[size];
        L[0] = 1;
        for (int i = 1; i < nums.length; i++) {
            L[i] = nums[i - 1] * L[i - 1];
        }

        int[] R = new int[size];
        R[size - 1] = 1;
        for (int i = size - 2; i >= 0; i--) {
            R[i] = nums[i + 1] * R[i + 1];
        }
        // 利用算好的左右乘积计算最终结果
        int[] ans = new int[size];
        for (int i = 0; i < nums.length; i++) {
            ans[i] = L[i] * R[i];
        }
        return ans;
    }


    /**
     * 方法二：空间复杂度 O(1)O(1) 的方法
     * 由于输出数组不被视为额外空间，因此可用输出数组来存储 L 或 R 的结果。
     * 先把输出数组当作 L 数组来计算，然后再动态构造 R 数组得到结果
     * <p>
     * 时间 O(n)，空间 O(1)
     *
     * @param nums nums
     * @return 除 nums[i] 之外其余各元素的乘积。
     */
    public int[] productExceptSelf2(int[] nums) {
        int size = nums.length;

        // 先用 ans[i] 来存储索引i左侧所有元素的乘积,从左到右
        int[] ans = new int[size];
        ans[0] = 1;
        for (int i = 1; i < nums.length; i++) {
            ans[i] = nums[i - 1] * ans[i - 1];
        }

        // 然后用R来存储右侧所有元素的乘积，从右往左
        int R = 1;
        for (int i = size - 1; i >= 0; i--) {
            ans[i] = ans[i] * R;
            // 更新右侧乘积
            R = nums[i] * R;
        }
        return ans;
    }

}
```
